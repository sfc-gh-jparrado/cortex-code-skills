# Configuración Previa en Snowflake

## 1. Crear Base de Datos y Schema

```sql
-- Crear database si no existe
CREATE DATABASE IF NOT EXISTS ${SNOWFLAKE_DATABASE};

-- Crear schema
CREATE SCHEMA IF NOT EXISTS ${SNOWFLAKE_DATABASE}.${SNOWFLAKE_SCHEMA};

-- Crear stage interno para archivos temporales
CREATE OR REPLACE STAGE ${SNOWFLAKE_DATABASE}.${SNOWFLAKE_SCHEMA}.AVRO_TMP
    FILE_FORMAT = (TYPE = 'AVRO');
```

## 2. Crear External Stage para S3 (Alternativa)

Si se prefiere usar un External Stage en lugar de escribir directamente:

```sql
-- Crear storage integration
CREATE OR REPLACE STORAGE INTEGRATION S3_INTEGRATION
    TYPE = EXTERNAL_STAGE
    STORAGE_PROVIDER = 'S3'
    ENABLED = TRUE
    STORAGE_AWS_ROLE_ARN = 'arn:aws:iam::${AWS_ACCOUNT_ID}:role/${ROLE_NAME}'
    STORAGE_ALLOWED_LOCATIONS = ('s3://${S3_BUCKET}/${S3_PREFIX}/');

-- Crear external stage
CREATE OR REPLACE STAGE ${SNOWFLAKE_DATABASE}.${SNOWFLAKE_SCHEMA}.S3_STAGE
    STORAGE_INTEGRATION = S3_INTEGRATION
    URL = 's3://${S3_BUCKET}/${S3_PREFIX}/'
    FILE_FORMAT = (TYPE = 'AVRO');
```

## 3. Crear Tabla Iceberg Destino

### Plantilla Genérica

```sql
CREATE OR REPLACE ICEBERG TABLE ${SNOWFLAKE_DATABASE}.${SNOWFLAKE_SCHEMA}.${TABLE_NAME} (
    -- Columnas de negocio (copiar del schema Oracle)
    ID NUMBER(38,0),
    COLUMN_VARCHAR VARCHAR(255),
    COLUMN_DATE DATE,
    COLUMN_TIMESTAMP TIMESTAMP_NTZ,
    COLUMN_NUMBER NUMBER(18,2),
    
    -- Columna de partición calculada (OBLIGATORIA)
    PARTITION_DATE DATE NOT NULL
)
CATALOG = 'SNOWFLAKE'
EXTERNAL_VOLUME = 'iceberg_external_volume'
BASE_LOCATION = '${TABLE_NAME}/'
PARTITION BY (PARTITION_DATE);
```

## 4. Generar Definición de Tabla desde Oracle

Para obtener el esquema de una tabla Oracle y convertirlo a Snowflake:

```sql
-- Ejecutar en Oracle para obtener columnas
SELECT 
    COLUMN_NAME,
    DATA_TYPE,
    DATA_LENGTH,
    DATA_PRECISION,
    DATA_SCALE,
    NULLABLE
FROM ALL_TAB_COLUMNS
WHERE OWNER = '${ORACLE_SCHEMA}'
  AND TABLE_NAME = '${ORACLE_TABLE}'
ORDER BY COLUMN_ID;
```

### Mapeo de Tipos Oracle → Snowflake

| Oracle Type | Snowflake Type |
|-------------|----------------|
| VARCHAR2(n) | VARCHAR(n) |
| CHAR(n) | CHAR(n) |
| NUMBER | NUMBER(38,0) |
| NUMBER(p,s) | NUMBER(p,s) |
| DATE | DATE o TIMESTAMP_NTZ |
| TIMESTAMP | TIMESTAMP_NTZ |
| CLOB | VARCHAR(16777216) |
| BLOB | BINARY |
| RAW | BINARY |

## 5. Permisos Necesarios

```sql
-- Grants para el rol de OpenFlow
GRANT USAGE ON DATABASE ${SNOWFLAKE_DATABASE} TO ROLE ${OPENFLOW_ROLE};
GRANT USAGE ON SCHEMA ${SNOWFLAKE_DATABASE}.${SNOWFLAKE_SCHEMA} TO ROLE ${OPENFLOW_ROLE};
GRANT SELECT, INSERT, DELETE ON ALL TABLES IN SCHEMA ${SNOWFLAKE_DATABASE}.${SNOWFLAKE_SCHEMA} TO ROLE ${OPENFLOW_ROLE};
GRANT USAGE ON STAGE ${SNOWFLAKE_DATABASE}.${SNOWFLAKE_SCHEMA}.AVRO_TMP TO ROLE ${OPENFLOW_ROLE};
GRANT USAGE ON WAREHOUSE ${SNOWFLAKE_WAREHOUSE} TO ROLE ${OPENFLOW_ROLE};
```

## 6. Verificar External Volume (para Iceberg)

```sql
-- Listar external volumes disponibles
SHOW EXTERNAL VOLUMES;

-- Describir un external volume
DESC EXTERNAL VOLUME iceberg_external_volume;

-- Crear external volume si no existe (requiere ACCOUNTADMIN)
CREATE OR REPLACE EXTERNAL VOLUME iceberg_external_volume
    STORAGE_LOCATIONS = (
        (
            NAME = 'my_s3_storage'
            STORAGE_PROVIDER = 'S3'
            STORAGE_BASE_URL = 's3://my-iceberg-bucket/iceberg/'
            STORAGE_AWS_ROLE_ARN = 'arn:aws:iam::${AWS_ACCOUNT_ID}:role/iceberg-role'
        )
    );
```
